## Instalation

You can install the package via composer:

```bash
composer require zaimea/accounts-sdk-laravel
```

or via composer.json

```json
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/zaimea/accounts-sdk-laravel"
        }
    ]
```
